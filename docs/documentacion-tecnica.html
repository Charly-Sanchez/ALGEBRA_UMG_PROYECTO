Algoritmos Implementados

Expansión de Laplace para Determinantes

Archivo: laplaceExpansion.ts

Complejidad: O(n!)

Implementación:

export function calculateDeterminantWithSteps(matrix: Matrix): {
  determinant: number;
  steps: Step[];
} {
  const steps: Step[] = [];
  
  function laplaceExpansion(mat: Matrix, level: number = 0): number {
    if (mat.length === 1) return mat[0][0];
    if (mat.length === 2) {
      return mat[0][0] * mat[1][1] - mat[0][1] * mat[1][0];
    }
    
    let det = 0;
    for (let j = 0; j < mat.length; j++) {
      const cofactor = getCofactor(mat, 0, j);
      const sign = (j % 2 === 0) ? 1 : -1;
      const minorDet = laplaceExpansion(cofactor, level + 1);
      
      det += sign * mat[0][j] * minorDet;
      
      steps.push({
        description: `Cofactor C₀${j} = ${sign === 1 ? '' : '-'}${mat[0][j]} × ${minorDet}`,
        matrix: mat,
        highlightedElements: [[0, j]],
        operation: `${sign === 1 ? '+' : ''}${sign * mat[0][j] * minorDet}`
      });
    }
    
    return det;
  }
  
  const determinant = laplaceExpansion(matrix);
  return { determinant, steps };
}

Características:

  • Recursivo con memoización
  • Generación de pasos detallados
  • Manejo de signos alternados
  • Optimización para matrices pequeñas

Eliminación Gaussiana

Archivo: gaussianElimination.ts

Complejidad: O(n³)

Proceso:

  1. Formación de matriz aumentada
  2. Eliminación hacia adelante
  3. Sustitución hacia atrás
  4. Verificación de solución

Manejo de casos especiales:

  • Pivoteo parcial para estabilidad numérica
  • Detección de sistemas inconsistentes
  • Identificación de variables libres
  • Manejo de división por cero

Matriz Inversa por Adjunta

Archivo: matrixInverse.ts

Fórmula: A⁻¹ = (1/det(A)) × adj(A)

Pasos del algoritmo:

  1. Calcular determinante
  2. Verificar invertibilidad (det ≠ 0)
  3. Calcular matriz de cofactores
  4. Transponer para obtener adjunta
  5. Dividir por determinante
  6. Verificar A × A⁻¹ = I

Visualización de cofactores:

function calculateCofactorMatrixDetailed(matrix: FractionMatrix): {
  cofactorMatrix: FractionMatrix;
  steps: InverseStep[];
} {
  const steps: InverseStep[] = [];
  const cofactorMatrix: FractionMatrix = [];
  
  for (let i = 0; i < n; i++) {
    cofactorMatrix[i] = [];
    for (let j = 0; j < n; j++) {
      const minorMatrix = getMinorMatrix(matrix, i, j);
      const minorDet = calculateFractionDeterminant(minorMatrix);
      const sign = ((i + j) % 2 === 0) ? 1 : -1;
      const cofactor = minorDet.multiply(new Fraction(sign));
      
      cofactorMatrix[i][j] = cofactor;
      
      steps.push({
        type: 'cofactor_calculation',
        description: `Cofactor C₍${i+1},${j+1}₎`,
        matrix: matrix,
        excludedRow: i,
        excludedCol: j,
        minorMatrix: minorMatrix,
        sign: sign,
        result: cofactor
      });
    }
  }
  
  return { cofactorMatrix, steps };
}

Instalación

1. Clonar el repositorio

git clone https://github.com/Charly-Sanchez/ALGEBRA_UMG_PROYECTO.git
cd ALGEBRA_UMG_PROYECTO

2. Instalar dependencias

npm install

3. Iniciar servidor de desarrollo

npm run dev

Navegar a http://localhost:5173
